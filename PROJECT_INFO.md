# Project Information

## Configuration
- **Backend Port:** 3020
- **Start Command:** `npm run server`

## Process Management
### PM2
If the application is running via PM2, you can manage it using:
- **Stop:** `pm2 stop <app_name_or_id>`
- **Restart:** `pm2 restart <app_name_or_id>`
- **List processes:** `pm2 list`
